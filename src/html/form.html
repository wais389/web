Pseudo-classes

root
Corresponds to the element that is the root of the document.
link
Corresponds to unvisited links.
visited
Corresponds to the visited links.
hover
Corresponds to the element that the mouse cursor is hovering over.
active
It is suitable when the user activates (for example, clicks the mouse) an element.
focus
Corresponds to the element that has focus.
Псевдоклассы группы child
Corresponds to an element that is somehow among other child elements of the same ancestor.
Псевдоклассы группы type
Corresponds to an element that is of a certain type among other child elements of the same ancestor.
checked
Corresponds to a radio button or checkbox in the selected state.
disabled, enabled
Corresponds to user interface elements that are in the disabled/enabled state.
required
Corresponds to the mandatory element of the form.

Pseudo-elements

first-letter
Corresponds to the first letter of the element.
first-line
Corresponds to the first line of the content of the generating element.
selection
Corresponds to the part of the document that was selected.
marker
Corresponds to the marker field of the list item, which usually contains a bold dot or number.
before
Corresponds to an element that allows styling and appears before the current content of the generating element.
after
Corresponds to an element that allows styling and appears after the current content of the generating element.
